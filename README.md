# Tun Tauk's Birthday Wish Website 🎂

A minimalist, developer-themed single-page website built for a programmer friend named **Tun Tauk**.

## Stack
- **HTML5** (semantic structure)
- **Vanilla CSS** (custom HSL colors, grid background, blur filters, macOS-style cards)
- **Vanilla JavaScript** (subtitle typing animation, custom HTML5 Canvas confetti generator, interactive console logging)
- **Vite** (dev server)

## Features
- **Modern Dark Dev UI**: HSL color scheme, glowing backdrops, and card-based code editor / terminal styling.
- **Typing Animation**: Simulates loading birthday wishes interactively.
- **Release Changelog**: Formatted as a version update (`vNewYear.0`) containing new features (happiness, success, etc.) and bug fixes (stress reduced, worries removed).
- **Run Celebration() Button**: Executes a physical Canvas-based confetti burst and logs random developer-centric birthday wishes to a console window output.

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the local server:
   ```bash
   npm run dev
   ```

3. Open the link displayed in the terminal (usually `http://localhost:5173`) in your browser.
